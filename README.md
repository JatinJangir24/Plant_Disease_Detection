# Plant_Disease_Detection

This is a repo for a basic setup for Plant Disease Detection uning TF-LITE on mobile devices. The Model was trained on PC using the python script "plant_disease_model.py".
The Trained Model HDF5 file and the converted tflite file is also provided.
The TF-LITE was used with a Flutter Android Application.
